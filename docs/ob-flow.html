Onboarding Canvas — Single Persistent Surface
OnboardingCanvasView is one single screen that never navigates. There is no NavigationStack.
VaylDirector owns a phase: OBPhase property. Calling director.advance(to:) is the only way to move forward.
Each phase is an overlay rendered on top of the same table canvas.
StatPhase Phase 1
StatPhase.swift
Opening screen. Research stat copy ("1 in 5..."), dealer line, CTA button to enter the table world.
Visual
Void background + OnboardingAtmosphere. Table not yet visible.
NamePhase Phase 2
NamePhase.swift
Table fades in. Dealer types a greeting. A card deals via SpriteKit physics, flips face-up. User types their name. Card pockets to corner deck.
Credential collected: name → deck[1]
Visual
SpriteKit CardFlightScene (arc deal), flip animation (scaleX 0→1), typewriter dealer line, TableSurfaceView fade.
Data
onboardingData.displayName written. Gender card placed silently (pendingGenderCard) for next phase.
ModeSelectPhase Phase 3
ModeSelectPhase.swift
Two cards deal face-down (mirror deal). User lifts one: Together, Solo, or Browsing. Selection card pockets.
Credential collected: mode → deck[2]
Visual
Mirror deal (two cards in sync). Tap-to-lift scaleEffect. Swipe-up confirmation.
Data
onboardingData.appMode set. Drives genderSpinCount (1=solo, 2=together) in next phase.
GenderPhase Phase 4
GenderPhase.swift
Card crystallises out of the felt (7-second dissolution sequence). Flips to reveal a slot machine face. User pulls handle → reels spin → drum picker appears. Swipe right to confirm. Together mode runs twice.
Credential collected: gender → deck[3]
Visual
dissolutionT drives 8 computed visual curves. SlotMachineCardFace: handle pull, reel spin, staggered settle, winning glow. Drum picker drag sync. Swipe-hint loop after reels settle.
Data
onboardingData.genderA / genderB written on confirm. Together mode shows partner handoff beat between spins.
ExperienceLevelPhase Phase 5
ExperienceLevelPhase.swift
Monte-style deal, cards shuffle, flip to reveal a candle face. User swipes up to select their intimacy experience stage.
Credential collected: experienceLevel → deck[4]
Visual
Monte shuffle animation (deckWeave). Candle card face illustration.
ContextPhase Phase 6
ContextPhase.swift
Cards deal face-up. Table fades. A carousel of emotional-context options slides in. User picks their current emotional register (e.g. Anxious, Excited). Swipe up to confirm.
Credential collected: context → deck[5]
Visual

Data

QuizPhase Phase 7
QuizPhase.swift
Four answer cards deal to the felt. One research-backed question floats above. User taps their answer. Correct card glows. All four vacuum-pull off to the right.
No credential — ephemeral, data used internally only
Visual
Four-card staggered deal (arc paths). Tap lifts selected card. Correct card spectrum bloom glow. Vacuum-pull exit to right edge simultaneously.
CuriosityPhase Phase 8
CuriosityPhase.swift
Tinder-style swipe. Cards come one at a time — swipe right to keep, left to pass. Two rounds with a dealer line in between. Final deck card pockets on round 2 completion.
Credential collected: curiosity → deck[6] — deck complete
Visual
Dealer line: "Sweep away what you aren't ready for." CuriosityStatusStrip, CuriosityPill. Swipe-right/left gesture with live rotation tracking.
Data
onboardingData.curiositySelections[]. Combined with emotionalRegister to set openerDeckType (anxious/excited).
BuildingPathPhase Phase 9
BuildingPathPhase.swift
All 6 credential cards fan out from the corner deck. User reviews them. Swipe right to accept and move to deck build.
Visual
deckFan animation from corner deck. Cards display credential faces. CornerDeckView pulses.
FoilPhase Phase 10
FoilPhase.swift
A foil card materialises on the table. Dealer types personalised copy. User taps the foil 3 times to tear it, revealing the opener deck underneath.
Visual
HolographicShimmer metal shader. foilIntegrity (1→0) drives tear dissolve. FoilTear tap tracking. foilDissolve animation token.
Data
