Path + Vault — 3 layout options

The floating Pulse hero is final — identical, pixel-for-pixel, in all three options below. What's actually undecided is what happens underneath it: right now Path and Vault are two separate full-width bordered cards stacked in a list. These three options change that relationship, not the hero.

1. Stacked triptych — today's arrangement, for reference. 2. Merged container — Path and Vault share one border, split by an internal seam. 3. Two-up row — Path and Vault sit side by side as a compact pair, so the hero is the only full-width element on the screen.

Hero, then Path (full-width card), then Vault (full-width card) — three separate surfaces in a vertical list. Included for reference, not as a real option.
