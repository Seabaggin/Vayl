Pulse hero — 3 more creative options

Round 2. The hero is still final and untouched. These three are structurally different from stacked/merged/row — they change how the hero's light relates to the content below it, not just its box shape.

4. Bleed-through — the hero's glow physically overlaps the top card. 5. Atmosphere merge — the glow becomes the screen's ambient wash; cards float on it. 6. Swipeable tray — Path and Vault become a horizontal carousel, hero stays the only static element.

Option 4

Bleed-through hero — light spills onto the card below

The hero's ambient glow is taller and the Path card is pulled up underneath it, so the glow physically washes over the card's top edge — like light spilling onto a surface, not a hairline seam connecting two shapes. Bolder continuity than a divider line, but riskier: it needs to read as intentional light, not a rendering artifact.

Option 5

Atmosphere merge — the whole screen carries the hero's light

Instead of a localized glow behind the orb, the lens color washes the entire screen — the same atmosphere language already used in Onboarding. Path and Vault go slightly glassier so the wash tints through them faintly. The hero stops being "a component with a glow" and becomes the light source for the whole screen. Most on-brand, but the biggest rework and a real risk to text contrast if pushed too far.

Option 6

Swipeable tray — Path and Vault become a carousel

Path and Vault sit in a horizontally-scrolling tray, each card sized to nearly fill the screen with the next peeking in at the edge. The hero is the only thing that's ever static and fully visible — everything else is one swipe away. Compact and modern, but a real discoverability risk: the peeking edge has to be enough to invite the swipe, or Vault effectively disappears.
