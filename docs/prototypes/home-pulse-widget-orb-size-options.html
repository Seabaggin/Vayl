The tension I flagged: this card now reads as a second hero next to Home's actual hero (the VaylCardFace). Home's dashboard was deliberately converged around one focal point with Pulse as a secondary, animated-placeholder signal beside it — this version competes for that role instead of supporting it. It also visually duplicates what the Map tab's Pulse hero already does (2D circumplex + living caustic aura), so you'd have two "big orb" moments in the app instead of one quick-glance + one immersive. Worth sitting this next to the real Home hero card before deciding — a mockup in isolation always looks more justified than it does next to the thing it's competing with.
