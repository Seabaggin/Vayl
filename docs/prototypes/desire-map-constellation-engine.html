How one engine covers 1 to 17 matches

Positions: a seeded golden-angle spiral, perturbed by the couple's id (rotation plus jitter). It spreads any count evenly with no crowding, and the seed makes each couple's sky unique.

Shape: a minimum spanning tree over the stars (always one connected figure, never a web) plus one or two short extra links for richness. A line draws once both its endpoints are lit, so the same logic works at any count.

Time and size: the per-star stagger shrinks as the count grows (fixed time budget), and star size scales down with count, so ten stars do not drag or crowd. 1 match is a lone star with no lines; 2 is a single link.

Variety: the three telegraphs only set the lighting order, so they compose freely with any count and any seed. No shape is ever drawn by hand.
