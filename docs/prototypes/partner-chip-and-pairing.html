4 · Two tiles + manage

Side-by-side quick-view tiles — Desire Map activity and Alex's current Pulse orb — each tapping through to the Map tab, plus a single Manage row underneath. This follows the same rule Pulse already ships under elsewhere in the app: openly viewable, gated only on Alex's own "share Pulse with partner" toggle, not on whether you've logged yours. If she has it off, the tile goes quiet instead of showing nothing (try the toggle below the frame).

Deliberately included: Pulse — it already ships elsewhere in the app as openly viewable (gated only on the owner's own share toggle, never on the viewer's own logging), so a glance of it here isn't new exposure, just a smaller window onto data Map already shows in full.
