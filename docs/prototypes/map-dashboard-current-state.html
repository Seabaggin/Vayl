The real question this raises: the Pulse hero is now borderless, breathing, ambient — clearly the "hero" register. Path and Vault are both still bordered glass cards. Is that contrast intentional hierarchy (one hero, two calmer utility cards below it — which is honestly how it reads to me, and matches "one thing breathes" from the original design principle), or does it now read as visually inconsistent — like the hero got a redesign pass and the other two didn't? I lean toward "intentional" — a screen with three equally-loud borderless elements would compete for attention, and Pulse earning the one un-boxed treatment is consistent with it being the single most-visited, most emotionally central piece. But worth your eyes on it before we lock Path's real design in.

What's real vs. proposed: the Pulse hero reflects the actual current build (no chrome, 0.62 aura ratio, ambient glow). My Vault doesn't exist in code yet, but is mocked here at full scale — same door treatment as Our Vault, cyan instead of magenta, "Your map · Your journal · only you" per the original reference concept — so the Me lens is a fair comparison, not a stub. Path is still the compact "pathfs" widget proposal from earlier this session, not yet built. The phone frame is now the realistic 320:745 ratio used elsewhere in this project's mockups, so the fit question below is an honest one.
