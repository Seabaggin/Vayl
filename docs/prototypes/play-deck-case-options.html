Play · static deck-case — Segment 2 options

The wall's perceived-empty problem was the old case rendering near-blank dark boxes (the 3D MetallicCaseView in flat mode loses its tilt-driven foil band). These are fully static (no shader, no per-frame work), so a whole wall stays cheap. Each deck's colorway (category spectrum slice + per-deck hue nudge) tints the metal + frame. Option A is what I built in Swift tonight; B and C are alternatives to redirect toward in the morning.
