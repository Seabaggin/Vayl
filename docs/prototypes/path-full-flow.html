The Path — the full flow

One converged sequence, not alternatives to pick between: the dashboard widget expands into the trail, a top-right toggle swaps to a list reading of the same data, tapping a node opens the real Mission Brief (unchanged) with a small overflow for the management actions, skipping actually removes the landmark, and everything removed lives in one Edit mode where it can come back. Solo and couple are the same flow with different framing, not different screens.

What changed from the earlier A/B pass: this isn't a choice between "trail everywhere" and "the ledger" anymore — it's both, as one screen with a view-toggle. The NodeView stopped being a competing invented sheet and became the real Mission Brief plus one small overflow menu.
