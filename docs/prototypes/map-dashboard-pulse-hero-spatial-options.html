Pulse hero — 3 real spatial layouts

Round 3. These actually rearrange the screen, not just relight the same vertical stack. Hero content (copy, orb, check-in pill) is unchanged; what changes is where Path and Vault live relative to it in space.

A. Split-zone — a real sky/ground break: hero alone up top, Path+Vault together in a distinct docked panel below. B. Orbit — Path and Vault shrink to small satellite moons beside the orb, not cards underneath it. C. Deck/fan — Path and Vault overlap as a fanned pair instead of a list.

Option A

Split-zone — a real horizon, not sequential cards

The screen splits into two zones: an open "sky" where the hero floats alone with real room to breathe, and a distinct "ground" panel below — its own tinted surface with rounded top corners, like a permanently-docked deck — holding Path and Vault together. A horizon line (with a soft wash of the lens color) marks the actual boundary, instead of the eye just scrolling from one card to the next.

Option B

Orbit / satellite — Path and Vault leave the list entirely

Path and Vault stop being cards underneath the hero and become two small satellite moons attached to the orb itself — reusing the constellation language from the Desire Map. Tapping a satellite is what opens that pillar. Below, a single quiet recap line replaces the two full preview cards. Most compact by far, but you give up Path's trail preview and Vault's stat line at a glance — the satellites can only carry a name.

Option C

Deck / fan — Path and Vault overlap, not stack

Path and Vault fan out like a hand of cards — Path in front (it's the live, changing one), Vault peeking behind at an angle (it's the static archive). Ties into the card-game visual identity already used in BuildDeck and sessions. Tapping the back card would bring it forward. Playful and compact, but whatever's in back is hidden by default, same discoverability trade-off as a carousel.
