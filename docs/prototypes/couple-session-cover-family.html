The .vaylCover shell — airlock → transition → close. Dismiss-disabled, confirm-on-exit. Same void + ember + spectrum vocabulary as the in-session hero, so the whole cover reads as one continuous, protected room.

#6 · airlock gesture — recommendation

Rules read aloud, not tapped — no per-rule checkboxes (that's chore-friction, not ritual). One tap on We're ready advances. The deliberate friction lives entirely in 1B's 3-sec press-and-hold — a sustained gesture you can't do absentmindedly, mirroring "take a breath." First session shows all six; repeats collapse to a one-line "settle in."

#8 · bandwidth form — recommendation

A 3-detent slider (Light · Open · Deep), not a continuous slider (false precision) nor a bare segmented control (loses the tactile "dial it in" gesture). Set privately; the session takes the gentler of the two as the depth ceiling — so it caps (honors "never push the unready," per the evidence base) but is never shown as a number to your partner.
